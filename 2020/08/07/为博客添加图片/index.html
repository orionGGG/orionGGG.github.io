Hexo中插入图片和音乐的几种方式

发表于 2020-08-07 更新于 2020-09-28

插入图片

方式一

首先修改主页配置文件_config.yml中的post_asset_folder这个选项的值为true。
在hexo目录git bush here并执行命令npm install hexo-asset-image –save，目的：下载安装一个可以上传本地图片的插件。该插件源自于：https://github.com/CodeFalling/hexo-asset-image

安装后，运行hexo new” “生成博文时，/source/_posts文件夹内除了md文件还有一个同名的文件夹
想要上传的图片复制到这个文件夹中，然后在md中引入图片：![你想要输入的替代文字](文件名/图片名.jpg)

![]()是markdown中常用的引用图片的格式，[]中是图片说明，()中写图片路径或者url
好处在于方便管理,每篇博客用到的图可以进行很好的归类,就是要下载个插件有点麻烦

方式二

![]()是markdown中常用的引用图片的格式，[]中是图片说明，()中写图片路径或者url

url为该图片的链接

将图片存放在source\images文件夹下,不用插件就能调用,缺点在于所有图片都存放在这里,调用数量多了易混淆

添加音乐

方式一

利用网易云音乐提供的外链播放器

点击音乐下方的“生成外链播放器”，然后进入外链生成界面，复制最下方的HTML代码，拷贝到自己想放的地方即可。
