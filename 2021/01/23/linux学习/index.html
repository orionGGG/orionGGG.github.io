linux学习

发表于 2021-01-23 更新于 2021-02-01

linux的学习之路

基本命令

路径及显示相关

cd

cd 变更路径 ——change directory的缩写
cd ..用来打开上一级路径

pwd

pwd 显示当前目录位置 ——print work directory的缩写

ls

ls 显示当前目录下的所有文件 ——list的缩写

  • -l以列表的形式返回
  • -a显示隐藏文件
  • -d查看目录属性
  • -l -a 显示所有的文件以列表形式返回
  • -l -h文件大小以KB来显示，原来是以字节来显示
    ls -al 和ls -la均为缩写
    ls -hl ls -lh都一样

通配符

*可以代表0个或多个任意字符 例如 ls *.txt就是找出所有含.txt的文件
?只可以代表1个字符 ls ??找出两个字符的文件
[] 可以代表中括号内的任意一个字符 [123]可以代表1或者2或者3
[1-3]与[123]等效
[a-z]表示a到z中的一个字符

注意事项

tab可以自动补全
输入的前几个字母对应不唯一的时候按两次tab反映所有的可能的文件

上下方向键可以取得历史命令(你以前输入过的命令)

clear 清空整个屏幕
cd ~可以回到根目录

文件及目录相关

操作文件

touch

创建空文件 touch 123.txt 创建隐藏文件 touch .yincang.txt
想要创建带空格文件得用双引号————> touch “program files”
不然的话会视作创建program和files两个文件
但并不推荐把空格作为文件名的储存部分，因为空格常用做分隔符

mkdir

创建目录(即创建文件夹) ——make directory的缩写

  • -p 递归创建，可一下子创建多级目录
rm

删除文件 ——remove的缩写
rm 命令可以删除一个目录中的一个或多个文件或目录，也可以将某个目录及其下属的所有文件及其子目录均删除掉
后面可接以下选项

  • -d 只能移除空文件夹
  • -f 强制删除文件或目录
  • -i 删除已有文件或目录之前先询问用户
  • -r 递归处理， 将指定目录下的所有文件与子目录一并处理
  • / 在Linux中表示根目录
  • * 所有文件
  • /* 根目录下的所有文件
mv

剪切文件、改名 ——move的缩写
mv 123.txt aa(aa就是目录，可以是相对路径(相对于当前目录)也可以是绝对路径)
mv 123.txt /home/gg/Desktop

cp

复制文件 ——copy的缩写

  • -r 复制目录
  • -p 保留文件属性(比如不想更改文件的修改时间)

显示文件内容

cat

显示文件内容

  • -n 显示行号(标出第几行)
  • tac 同cat不过是从最后一行往前
more

分页显示文件内容(如果一个文件的内容很长就用more)
进入到more浏览状态后可以使用

  • (空格)或f 翻页
  • (回车) 换行
  • q或Q 退出
less

分页显示文件内容(比more多了一个功能就是可以向上翻)
用pageup一页一页往上翻，用 ↑ 一行一行往上翻

head

显示文件前几行
head -n 7 文件名 ——显示前7行
head 文件名 ——默认显示前10行

tail

显示文件末尾几行

  • -n 指定行数
  • -f 动态显示文件末尾内容

链接命令

ln

ln ——link的缩写

  • -s 创建软链接

ln -s [原文件][目标文件]
目标文件就是新生成的链接文件

软链接的特征：类似Windows的快捷方式
只是符号链接，大小很小，链接箭头指向原文件

硬链接的特征：拷贝cp -p + 同步更新(对原文件的处理也会反应在目标文件上)
硬链接相当于两条路径指向了同一个文件，删除一个路径下的文件并非删除了文件，而是删除了路径对文件的指向
通过i(inode)节点识别
不能跨分区
不能针对目录使用

权限管理命令

chmod

chmod ——change the permissions mode of a file的缩写

只有所有者u和管理员root可以更改文件及目录权限

chmod [{ugoa}{+-=}{rwx}][文件或目录] chmod g+w,o-r,a=rwx test.list

  • u 表示所有者、g表示所属组、o表示其他人、a表示所有人
  • + 表示增加权限;-表示减少权限;=表示老子不管你以前是什么鸟权限，反正现在我就要让它是这样的权限
  • Linux的权限不是很细致，只有RWX三种：
  • r(read，读取)：对文件而言，可以查看文件内容；对目录来说，可以列出目录中的内容。
  • w(write,写入)：对文件而言，具有新增,修改文件内容的权限(但不含删除该文件)；对目录来说，具有新建，删除，修改，移动目录内文件的权限。
  • x(execute，执行)：对文件而言，具有执行文件的权限；对目录了来说，该用户可以进入目录。

chmod [mode=421][文件或目录] chmod 627 test.list

  • r–4 2^2

  • w–2 2^1

  • x–1 2^0

  • rwxrw-r– 7 6 4

  • 三个数字对应ugo的状态

  • -r 递归修改(改变目录权限的同时改变目录下的所有文件)

a

在linux中以.开头就是隐藏文件，你改个名字就成了
